# Data Visualization Module

This module provides visual analytics for the Autonomous Code Reviewer & Security Auditor system.

## Objective
To help developers quickly understand:
- Which files are most risky
- What types of vulnerabilities are common
- Severity distribution
- Line-level vulnerability details

## Visualizations Implemented

1. File Risk Score (Bar Chart)
Shows total risk score for each file.

2. Vulnerability Distribution (Pie Chart)
Displays distribution of vulnerability types.

3. Severity Levels (Bar Chart)
Shows count of High, Medium, and Low severity issues.

4. Detailed Findings (Table)
Shows file name, line number, vulnerability type, severity, and explanation.
